### Coding along with Architecture Patterns with Python
The default branch, `main`, was created from the `start` tag.

## Running the tests

```sh
make test
# or, to run individual test types
make unit
make integration
make e2e
# or, if you have a local virtualenv
make up
pytest tests/unit
pytest tests/integration
pytest tests/e2e
```
